课程信息

尚品甄选，是一个以京东商城为参考原型的互联网 B2C 的电子商城项目，项目包含平台管理端、店铺管理端、商品 PC 端及微信小程序端。
有一套完善的电商业务流程：商品管理、商品检索、商品详情、购物车、单点登录、订单、支付、促销活动、优惠券、库存管理等，其中覆盖了分布式文件系统、全文检索数据库、高速缓存、消息队列等多种业务场景和技术实现。

如果你已经学习了 Java 基础、SSM 框架、SpringBoot、SpringCloud，想找一个项目来实战练习；或者你刚刚入行，需要可以写到简历中的微服务架构项目，本套项目教程就是为你量身打造！

这是一套适合新手入门的 SpringBoot+SpringCloud 企业级项目，项目采用前后端分离开发模式，使用了当前主流的开发技术，技术栈广且版本新，涵盖：JDK17、SpringBoot3、SpringCloud2022、Redis7、MySQL8 等。

教程延续了尚硅谷一贯的授课风格，从 0 到 1 保姆式讲解，配套资料齐全：视频、代码、课件、图示、资料，败家式赠送，从功能分析、编码实现、功能测试到 bug 解决，带领你从项目实际出发，一步步搭建出真正属于自己的微服务架构项目！
